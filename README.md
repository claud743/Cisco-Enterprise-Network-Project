Cisco Enterprise Network Project

This project simulates a real enterprise network environment using Cisco technologies.  
It was built in Cisco Packet Tracer and includes routing, switching, security, VPN connectivity, and enterprise services.


Network Overview

The network consists of two sites:

- Head Office (HQ)
- Branch Office

Both sites are connected through an ISP network and secured using an IPsec Site-to-Site VPN tunnel.


Technologies Implemented

Layer 2 Switching
- VLAN segmentation
- Spanning Tree Protocol (PVST)
- PortFast
- BPDU Guard

Layer 3 Routing
- Inter-VLAN Routing
- Dynamic Routing using OSPF

Network Services
- DHCP server
- Web Server (HTTP)
- TFTP Server (configuration backup)
- Syslog Server (centralized logging)

Security
- Access Control Lists (ACL)
- Guest VLAN isolation
- NAT for internet access
- Site-to-Site **IPsec VPN**

High-Level Network Features
- Secure communication between HQ and Branch
- Internet access via NAT
- Centralized logging and monitoring
- Secure configuration backup using TFTP
